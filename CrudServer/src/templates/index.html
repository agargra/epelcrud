Node js CRUD Server

API Endpoints

Mongo Endpoints

  • (GET /api/mongo/) Listado de elementos
  • (GET /api/mongo/:id) Ver elemento
  • (POST /api/mongo/create) Añadir elemento
  • (PUT /api/mongo/update/:id) Actualizar elemento
  • (POST /api/mongo/delete/:id) Eliminar elemento

Mysql Endpoints

  • (GET /api/mysql/) Listado de elementos
  • (GET /api/mysql/:id) Ver elemento
  • (POST /api/mysql/create) Añadir elemento
  • (PUT /api/mysql/update/:id) Actualizar elemento
  • (POST /api/mysql/delete/:id) Eliminar elemento
